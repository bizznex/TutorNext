# BizzNexx Learning Studio Beta Review Guide

Use this guide for selected tutor beta reviews. The goal is not to pitch everything. The goal is to learn whether the Learning Studio offer feels useful, trustworthy, and worth paying for.

## Review Setup

Target reviewer:

- Tutor, coach, or small learning studio owner
- Ideally teaches classes 8-12 or recurring skill batches
- Uses WhatsApp for parent/student inquiries
- Has trial, demo, or first-call workflow
- Collects monthly or batch fees

Session length:

- 30-45 minutes

Required:

- Beta URL
- Access code
- Screen sharing
- Notes document
- Feedback form

## Opening Script

"We are testing an early version of BizzNexx for tutors and learning studios. It is not a learning app or ERP. The idea is to help you look credible to parents, capture better inquiries, follow up on trials, and track monthly fees without maintaining a heavy CRM."

"Please react honestly. We are trying to find what is useful, what is confusing, and whether this is something a real tutor would pay for."

## Demo Flow

### 1. Parent Page

Show:

- Ananya Learning Studio
- Beta Studio tab
- Parent-facing hero
- Teacher profile
- Proof items
- Course cards
- FAQ

Ask:

- Would you share a page like this with parents?
- What would make it feel more credible?
- What feels fake, generic, or unnecessary?
- Would parents understand class, board, fee, timing, and trial clearly?

### 2. Trial Inquiry

Submit a sample inquiry:

- Name: Sensecheck Parent
- Phone: 9000001111
- Course: Class 9-10 Maths Foundation
- Class/board: Class 10 ICSE
- Concern/timing: Maths weak in algebra, weekday evening preferred

Ask:

- Are these the right fields?
- What do parents usually forget to mention?
- Is this better than a raw WhatsApp message?
- Is any field too much?

### 3. Owner Action Queue

Show:

- Parent-trial action queue
- Lead action
- Suggested WhatsApp message
- Status change

Ask:

- Would this help you remember whom to reply to?
- Does the language feel natural?
- Would you use "Trial options sent", "Trial scheduled", and "Fee plan sent"?
- What action is missing?

### 4. Monthly Fee Actions

Show:

- Pending payments
- Overdue ageing
- Payment reminder preview
- Mark paid action

Ask:

- Is fee tracking useful?
- Does the reminder feel polite enough?
- What makes fee reminders awkward today?
- Would this reduce awkwardness or just add work?

### 5. Beta Terms

Show:

- Beta terms page
- Safety note

Ask:

- Is the scope clear?
- Do you understand that BizzNexx tracks status but does not process payments?
- Is anything concerning about data/privacy?

## Willingness-To-Pay Questions

Ask directly:

- If we set this up for your tutoring business, what would you expect to pay once?
- Would INR 1,499-2,999 setup feel reasonable?
- Would INR 499-999/month feel reasonable for reminders, updates, and support?
- What outcome would make it worth paying for?
- What would make you stop using it?

## Scoring

After the session, score 1-5:

- Share-worthiness
- Parent trust
- Inquiry usefulness
- Trial follow-up usefulness
- Fee tracking usefulness
- Ease of use
- Willingness to pay
- Overall fit

## Success Signal

Strong signal:

- Tutor says they would share the page
- Tutor identifies real current pain
- Tutor sees value in trial/follow-up/payment actions
- Tutor gives specific pricing reaction
- Tutor asks how to customize their own page

Weak signal:

- Tutor sees it as "just a website"
- Tutor expects guaranteed leads
- Tutor dislikes maintaining statuses
- Tutor does not collect recurring fees
- Tutor cannot imagine using it weekly

## Close

"Thank you. The most useful feedback is where this does not fit your real workflow. We will use this to decide what to fix before inviting more beta users."
