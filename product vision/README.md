## Brand Architecture

Current recommendation:

- **ContexureEarth** is the public education and learning-intelligence brand.
- **ContexureEarth Tutor** is the first market product.
- **BizzNexx** is the existing operating layer/prototype foundation that should be reused and gradually made education-native.

## Product Spine

The first product should focus on:

```text
Tutor trust and inquiry
-> trial/demo conversion
-> fee discipline
-> proof and referrals
-> business analytics
-> learner context
-> learning intelligence
-> institutions
```

## Do Not Build First

Avoid these until the tutor wedge is validated:

- Full LMS
- Parent/student mobile app
- Marketplace lead generation
- Heavy coaching ERP
- Payment gateway dependency
- GST/accounting suite
- Automated WhatsApp API as the first workflow
